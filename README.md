# io-hexo-src
hexo静态网站源文件 http://kuangpeng.github.io

## Installation

``` bash
$ npm install
```

## start write
``` bash
$ hexo new (layout) project-name
```

## local server
``` bash
$ hexo server
```
浏览器打开 http://localhost:4000/ , 默认4000端口

## clean cache
``` bash
$ hexo clean
```
